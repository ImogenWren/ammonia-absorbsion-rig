Bronkhorst FLOW-BUS Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturers: Bronkhorst Cori-Tech, Bronkhorst High-Tech, and M+W Instruments.
Supported Language(s): LabVIEW
Supported Model(s): Series EL-FLOW, IN-FLOW, LOW-DP-FLOW, IQ+FLOW, MASS-VIEW, mini CORI-FLOW, CORI-FLOW, MASS-STREAM, EL-PRESS, IN-PRESS, LIQUI-FLOW, MU-FLOW.
Model(s) Tested: EL-FLOW (F-201C-RAD, F-111B-RAD), EL-PRESS (P-506C-RAA)
Interface(s): Serial (Direct or with FLOW-BUS to RS232 Converter)
Firmware Revision(s) Tested: V7.11; V7.25

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 1.2
Original Release Date: 01/20/2014
Current Revision Date: 12/19/2018
Current Release Date: 02/19/2019

2. Required Software

Some software components need to be installed before using this instrument driver. Minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 5.0 or later.

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.
